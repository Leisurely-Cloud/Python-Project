# Python-Project

一个使用 Python 编写的基础计算器项目，包含图形界面程序与测试代码。  
这个仓库记录了我学习 Python、练习项目开发和使用 GitHub 进行版本管理的过程。

## 项目介绍

本项目目前主要包含两个部分：

- `calculator_gui.py`：计算器图形界面程序
- `test_calculator_logic.py`：计算器逻辑相关测试代码

这是一个面向入门学习的小项目，目标是通过实际编写代码，熟悉以下内容：

- Python 基础语法
- 图形界面程序开发
- 项目结构整理
- 使用 Git 和 GitHub 管理代码
- 编写和运行测试

cd Python-Project

python calculator_gui.py
如何测试

如果你已经安装了 pytest，可以使用下面的命令运行测试：

pytest

如果没有安装 pytest，可以先执行：

pip install pytest

然后再运行：

pytest
